# Cutter

## 설명
로스트아크의 세공 시스템을 구현한 플러그인입니다.

## 명령어
- /세공열기 - 세공하기 위한 인벤토리를 오픈합니다.
- /세공설정 <player> <var> - 플레이어 개인의 세공 확률 업을 설정합니다.
- /돌제작 - `/세공열기` 에서 세공 가능한 아이템을 제작합니다.
